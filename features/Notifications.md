# Notifications

## Objective
Inform users of request activity without creating noise.

## Scope
- New request emails
- Acceptance emails
- Expiry emails
- Optional digests

## Constraints
- Email only
- No real-time push
- Respect user preferences

## Deliverables
- Email templates
- Trigger logic

## Acceptance Criteria
- Notifications sent reliably
- No duplicate emails
- Digest optional
